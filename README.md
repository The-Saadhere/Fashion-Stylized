# 🛍️ Fashion Stylized

A modern full-stack eCommerce web application built to deliver a fast, scalable, and user-friendly shopping experience. **Fashion Stylized** focuses on selling premium fashion accessories such as watches, wallets, and glasses with a clean UI and optimized performance.

---

## 🚀 Features

- Modern and responsive UI/UX  
- Secure authentication using NextAuth  
- Product listing and detailed product pages  
- Shopping cart functionality  
- State management with Redux / Zustand  
- Email integration using Resend  
- MongoDB database integration  
- Server-side rendering with Next.js  

---

## 🧰 Tech Stack

- Frontend: Next.js, React.js, Tailwind CSS  
- Backend: Next.js API Routes  
- Authentication: NextAuth  
- Database: MongoDB  
- State Management: Redux / Zustand  
- Email Service: Resend
